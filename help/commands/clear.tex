\doc{The clear command}
\ref{clear-command}

The \def{clear command} resets the entire state of LP.  

\head{2}{\dlink{../symbols/syntax}{Syntax}}
\begin{verbatim}
\sd{clear-command} ::= \f{clear}
\end{verbatim}

\head{2}{Examples}
\begin{verbatim}
clear
\end{verbatim}

\head{2}{Usage} 
The \fq{clear} command causes LP to discard all information except for the
values of the global \dlink{set}{settings}.
