\doc{Convergence}
\ref{convergence}
\ref{convergent}

A set of \llink{rewrite-rule}{rewrite rules} is called \def{convergent} if all
terms can be rewritten to a terminal form in a finite number of steps, and all
terms have a unique \olink{normalization}{normal form}.
